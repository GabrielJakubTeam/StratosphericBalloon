# StratosphericBalloon
Hardware stratospheric balloon localization module with server. 

# Hardware
- SIM7000E LTE GPS HAT - Waveshare 14865 
- UPS HAT - Waveshare 18306
- SIM card
- Raspbery Pi 4B
- Server
